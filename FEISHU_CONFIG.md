# 飞书多维表格 K线图查看器 - 配置指南

## 概述

本系统实现以下流程：

```
飞书多维表格[按钮列]
    ↓ 用户点击
自动化流程触发
    ↓ 读取当前行字段
    ↓ [发送HTTP请求] GET /api/chart?code={股票代码}&date={买入日期}
    ↓ [302重定向]
    ↓ [30分钟K线图页面打开]
```

## 文件结构

```
feishu_view_html/
├── index.html      # 前端K线图页面
├── server.js       # Node.js中转服务
├── package.json    # 依赖配置
└── FEISHU_CONFIG.md # 本配置文件
```

## 部署步骤

### 1. 安装依赖

```bash
cd feishu_view_html
npm install
```

### 2. 启动服务

```bash
npm start
# 或直接运行
node server.js
```

服务默认在 `http://localhost:3000` 启动。

### 3. 公网访问（重要）

飞书自动化需要在**公网可访问的URL**，你需要将服务暴露到互联网。推荐方案：

- **方案A**: 使用 ngrok 等内网穿透工具
- **方案B**: 部署到云服务器（阿里云、腾讯云等）
- **方案C**: 使用 Vercel/Cloudflare Workers 等平台

示例 - 使用 ngrok：
```bash
ngrok http 3000
# 获得公网URL，如 https://abc123.ngrok.io
```

## 飞书多维表格配置步骤

### 第一步：创建/准备多维表格

确保你的多维表格包含以下字段：

| 字段名 | 字段类型 | 说明 |
|--------|----------|------|
| 股票代码 | 文本 | 6位数字，如 600519 |
| 买入日期 | 日期 | 格式 YYYY-MM-DD |
| 查看K线 | 按钮 | 用于触发自动化流程 |

### 第二步：创建按钮字段

1. 打开多维表格
2. 点击右上角「+」添加新字段
3. 选择字段类型为 **「按钮」**
4. 命名为「查看K线」（或其他你喜欢的名称）

### 第三步：创建自动化流程

1. 打开多维表格，点击右下角 **「自动化」** 进入自动化编辑器
2. 点击「新建流程」
3. **触发条件**：选择 **「按钮被点击」**
   - 选择你刚创建的按钮字段

4. **执行动作**：添加一个 **「发送HTTP请求」** 动作
   - 请求方法：**GET**
   - URL：`https://your-domain.com/api/chart?code={{股票代码}}&date={{买入日期}}`
   
   > ⚠️ 将 `your-domain.com` 替换为你的实际公网域名
   
   > `{{股票代码}}` 和 `{{买入日期}}` 是飞书自动化中的变量引用，会动态填入当前行的值

5. **保存并启用**自动化流程

### 第四步：测试

1. 在多维表格中找到一条记录
2. 确认该记录有「股票代码」和「买入日期」的值
3. 点击「查看K线」按钮
4. 如果配置正确，会自动跳转到30分钟K线图页面

## API文档

### 302 Redirect端点

```
GET /api/chart?code={股票代码}&date={交易日期}
```

**参数说明**：
| 参数 | 类型 | 必填 | 说明 |
|------|------|------|------|
| code | string | 是 | 6位股票代码，如 600519 |
| date | string | 是 | 交易日期，格式 YYYY-MM-DD |

**响应**：
- 状态码：`302 Found`
- Location头：`/?code={code}&date={date}`

**示例**：
```
GET /api/chart?code=600519&date=2024-01-15
↓ 302 Redirect
/?code=600519&date=2024-01-15
```

## K线图功能说明

### 数据源

使用 **东方财富API** 获取实时股票数据，支持：
- A股（沪市、深市、北交所）
- 30分钟K线（分时K线）

### 图表特性

- 标准K线图（蜡烛图）
- 成交量柱状图
- 交互功能：缩放、拖动、悬浮查看详细信息
- 自动识别交易所后缀（.SH/.SZ/.BJ）

### 日期范围

点击按钮后，会显示**买入日期前后各一天**的30分钟K线图：
- 起始日期 = 买入日期 - 1天
- 结束日期 = 买入日期 + 1天

## 常见问题

### Q: 图表无法加载？

1. 检查网络连接
2. 确认股票代码格式正确（6位数字）
3. 确认日期格式为 YYYY-MM-DD
4. 检查服务器是否正常运行

### Q: 飞书自动化无法访问服务？

确保你的服务可以通过公网访问：
- 使用 ngrok 或其他内网穿透工具
- 部署到云服务器
- 测试：在浏览器中直接访问 `https://your-domain.com/api/test`

### Q: 如何自定义端口？

```bash
PORT=8080 npm start
```
